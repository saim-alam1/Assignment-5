Question-1: What are the different ways to select an element in the DOM?

There are different kinds of ways to select an element in the DOM. Such as getElementById(), getElementByClassName(), querySelector(), querySelectorAll(). Even we can call them by builting custom functions etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is "innerHtml" means all the html tags inside that element on the other hand innerText means only the texts inside the element and at last textContent means is a element which is type of container or box which which contains texts.

Question-3: What is event delegation in the DOM?

"event delegation" in the DOM is a way or technique where if we want to do something with the child elements or we wants something to happen if we click on them we dont need to add addEventListener() to every one of them we just target the parent element or the parent container and add the addEventListener() on them. In this way we dont need to write extra codes and it is a good practise.

Question-4: What is event bubbling in the DOM?

Event Bubling is a built in property in JavaScript. It starts when a event starts and goes all aloung to its parent element by bubling.

Question-5: How do you create, add, and remove elements using JavaScript?

First of all To create a element by JavaScript we have to use .createElement() method. On the other hand to add an element we use .appendChild() method. At last to remove any element there is two way to remove one .remove to remove the element permanently and the other classList.remove() which just removes the class name of the element.
